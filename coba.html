Cara Penggunaan

1. Pilih Mode: Gunakan tombol tab di atas untuk beralih antara Mode Musik atau Mode Murottal.

2. Tambah File: Klik tombol "TAMBAH" untuk memasukkan file MP3/MP4 dari folder HP/PC Anda.

3. Simpan Otomatis: File yang Anda tambahkan akan tersimpan di browser ini meski halaman di-refresh.

4. Hapus Lagu: Klik ikon silang (x) pada daftar lagu untuk menghapus satu per satu.
